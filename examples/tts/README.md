# TTS (Text-to-Speech) Example

This example demonstrates how to use Sarvam AI's Text-to-Speech capabilities to narrate a book summary.

### Files
- `book_summary_narrator.ipynb`: A Jupyter Notebook that takes a book summary and converts it into a synthesized audio narration.

### Usage
Run the `book_summary_narrator.ipynb` notebook to see the text-to-speech process in action to generate a high-quality narration from text.
