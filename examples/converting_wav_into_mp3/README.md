# Convert WAV to MP3

This example demonstrates how to convert a `.wav` audio file into a `.mp3` format using Python. This is especially useful for audio files obtained from or to be used with the Sarvam AI APIs.

### Files
- `wav_to_mp3.ipynb`: A Jupyter Notebook containing the code to perform the conversion.
- `output_file.mp3`: Placeholder for the output file.

### Usage
Run the Jupyter notebook `wav_to_mp3.ipynb` to see the step-by-step process of converting WAV files to MP3 format.
