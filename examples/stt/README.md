# STT (Speech-to-Text) Example

This example demonstrates how to use the Sarvam AI Speech-to-Text (STT) capabilities to generate lyrics from a song.

### Files
- `Song_Lyrics_Generator.ipynb`: A Jupyter Notebook that takes an audio file of a song and generates lyrics using Sarvam AI's STT model.

### Usage
Run the `Song_Lyrics_Generator.ipynb` notebook to see an end-to-end example of generating text from an audio file.
