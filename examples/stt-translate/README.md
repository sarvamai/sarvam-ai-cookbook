# STT Translate Example

This example showcases how to use Sarvam AI's Speech-to-Text translation features to automatically generate translated subtitles for audio or video content.

### Files
- `Subtitles.ipynb`: A Jupyter Notebook demonstrating the generation of translated subtitles from an audio source using Sarvam AI.

### Usage
Open the `Subtitles.ipynb` notebook and follow along to learn how to create seamless subtitle transcriptions.
